# 학생 대시보드 (My Library) - 사용 가이드

## 📚 개요

학생 대시보드는 여러분의 개인 학습 라이브러리입니다. 모든 수업 자료를 한눈에 보고, 원하는 수업을 빠르게 찾아 복습할 수 있습니다.

## ✨ 주요 기능

### 1. **웰컴 배너**
```
✨ 나의 학습 라이브러리
총 12개의 수업이 있습니다. 오늘도 화이팅하세요! 💪
```
- 전체 수업 개수 표시
- 응원 메시지로 학습 동기 부여

### 2. **수업 달력 (Calendar Widget)**
```
📅 한눈에 보는 수업 일정
- 수업이 있는 날에 파란 점 표시
- 날짜 클릭으로 해당 날짜 수업 필터링
- 모바일에서 접기/펼치기 가능
```

#### 사용 방법
1. **날짜 선택**: 원하는 날짜를 클릭
2. **수업 확인**: 해당 날짜의 수업만 표시됨
3. **필터 해제**: "날짜 필터 해제" 버튼 클릭

#### 달력 표시
- 🔵 **파란 점**: 수업이 있는 날
- **굵은 글씨**: 오늘 날짜
- **선택된 날짜**: 파란색 배경

### 3. **검색 기능**
```
🔍 수업 제목이나 내용으로 검색...
```

#### 검색 예시
- "인수분해" → 인수분해 관련 수업 찾기
- "2차 방정식" → 2차 방정식 수업 찾기
- "중간고사" → 중간고사 관련 수업 찾기

#### 검색 팁
- 키워드를 간단하게 입력
- 제목과 설명 모두 검색됨
- 실시간으로 결과 업데이트

### 4. **정렬 옵션**
```
📊 최신순 / 과거순
```

- **최신순** (기본): 최근 수업부터 표시
- **과거순**: 오래된 수업부터 표시

### 5. **수업 카드**
```
┌─────────────────────────┐
│   [썸네일 이미지]        │  ← 첫 번째 칠판 이미지
│   [1월 16일 (금)]        │  ← 날짜 배지
│   [NEW]                  │  ← 3일 이내 업로드
├─────────────────────────┤
│ 고1 수학 상 - 인수분해   │  ← 수업 제목
│ 곱셈 공식을 이용한...    │  ← 설명
│ 📖 12페이지  🎥 1개 영상 │  ← 자료 개수
└─────────────────────────┘
```

#### 카드 구성 요소
1. **썸네일**: 수업의 첫 번째 칠판 이미지
2. **날짜 배지**: 수업 날짜 (요일 포함)
3. **NEW 배지**: 최근 3일 이내 업로드된 수업
4. **제목**: 수업 제목 (2줄까지 표시)
5. **설명**: 수업 설명 (있는 경우)
6. **통계**: 페이지 수와 영상 개수

#### 카드 상호작용
- **클릭**: 전체 카드 클릭하면 E-Book 뷰어로 이동
- **호버** (마우스 올리기): 
  - 썸네일 확대
  - "복습하기 →" 버튼 표시
  - 제목 색상 변경

### 6. **활성 필터 표시**
```
활성 필터: [검색: "인수분해" ×] [날짜: 1월 15일 ×]
```

- 현재 적용된 필터 표시
- × 버튼으로 개별 필터 제거
- "모든 필터 해제" 버튼으로 한 번에 제거

## 🎮 사용 시나리오

### 시나리오 1: 특정 날짜 수업 찾기
```
1. 달력에서 원하는 날짜 클릭
2. 해당 날짜의 수업 카드만 표시됨
3. 카드 클릭하여 복습
```

### 시나리오 2: 주제로 수업 검색
```
1. 검색창에 "인수분해" 입력
2. 관련 수업만 필터링됨
3. 원하는 수업 선택
```

### 시나리오 3: 오래된 수업 복습
```
1. 정렬을 "과거순"으로 변경
2. 맨 위에 오래된 수업 표시
3. 순서대로 복습
```

### 시나리오 4: 최근 수업 확인
```
1. 대시보드 접속 (기본 최신순)
2. NEW 배지가 있는 수업 확인
3. 새로 올라온 자료 복습
```

### 시나리오 5: 복합 필터 사용
```
1. 달력에서 특정 주 선택
2. 검색창에 키워드 입력
3. 원하는 수업 정확히 찾기
```

## 📱 모바일 사용법

### 모바일 최적화 기능
```
✓ 카드 1열 배치
✓ 달력 접기/펼치기
✓ 터치 친화적 버튼
✓ 반응형 이미지
```

### 모바일 사용 팁
1. **달력**: 헤더를 탭하여 접기/펼치기
2. **검색**: 키보드 자동으로 나타남
3. **스크롤**: 부드럽게 스크롤하며 탐색
4. **카드**: 카드를 탭하면 바로 뷰어로 이동

## 💡 학습 팁

### 효과적인 복습 전략

#### 1. **주기적 복습**
```
Day 1: 수업 당일 복습
Day 3: 3일 후 복습 (NEW 배지 확인)
Day 7: 1주일 후 복습
Day 30: 1개월 후 복습
```

#### 2. **테마별 복습**
```
1. 검색으로 비슷한 주제 모으기
2. 한 번에 연속으로 복습
3. 개념 연결하며 학습
```

#### 3. **시험 대비 복습**
```
1. 과거순으로 정렬
2. 시험 범위 수업 순서대로 복습
3. 약한 부분 집중 학습
```

#### 4. **날짜별 복습**
```
1. 달력으로 특정 주 선택
2. 그 주의 모든 수업 복습
3. 전체적인 흐름 파악
```

## 🎨 UI 요소 상세

### 색상 코드
- **파란색**: 수업 있는 날짜, 선택된 요소
- **빨간색**: NEW 배지
- **회색**: 비활성 요소
- **그라데이션**: 웰컴 배너 (파랑→보라)

### 아이콘 의미
- 📖 **BookOpen**: 페이지 수
- 🎥 **Video**: 영상 개수
- 📅 **Calendar**: 날짜/달력
- 🔍 **Search**: 검색
- ✨ **Sparkles**: 특별한 요소
- ⬇️ **TrendingDown**: 최신순
- ⬆️ **TrendingUp**: 과거순

### 애니메이션
- **호버 효과**: 카드가 살짝 확대
- **NEW 배지**: 깜빡이는 효과
- **로딩**: Skeleton UI로 부드러운 전환
- **필터**: 실시간 업데이트

## 🔧 기술 사양

### 데이터 로딩
```javascript
✓ 썸네일: 첫 번째 칠판 이미지 (order=1)
✓ 페이지 수: blackboard_image 타입 materials 개수
✓ 영상 수: video_link 타입 materials 개수
✓ NEW 배지: 3일 이내 created_at
```

### 성능 최적화
```
✓ Skeleton UI로 로딩 상태 표시
✓ 이미지 지연 로딩 (Next.js Image)
✓ 클라이언트 사이드 필터링 (빠른 응답)
✓ useMemo로 불필요한 재계산 방지
```

### 반응형 그리드

## 🐛 문제 해결

### Q: 수업이 표시되지 않아요
```
A:
1. 인터넷 연결 확인
2. 페이지 새로고침 (F5)
3. 선생님께 자료 업로드 확인
```

### Q: 검색이 작동하지 않아요
```
A:
1. 검색어 철자 확인
2. 너무 긴 검색어는 짧게
3. 필터 해제 후 다시 검색
```

### Q: 달력에 파란 점이 안 보여요
```
A:
1. 해당 날짜에 수업이 없을 수 있음
2. 다른 달로 이동해보기
3. 페이지 새로고침
```

### Q: NEW 배지가 사라졌어요
```
A:
- 3일 이상 지나면 자동으로 사라짐
- 정상 동작입니다
```

### Q: 카드 이미지가 안 보여요
```
A:
1. 이미지가 없는 수업일 수 있음
2. 기본 아이콘이 표시됨
3. 선생님께 확인
```
