# 관리자 업로드 대시보드 - 사용 가이드

## 🎯 개선된 주요 기능

### 1. 좌우 분할 레이아웃
- **좌측**: 학생 선택 사이드바
  - 검색 기능으로 빠른 학생 찾기
  - 선택된 학생 강조 표시
  - 최근 업로드 날짜 표시 (추후 추가 예정)

- **우측**: 수업 자료 업로드 폼
  - 수업 정보 입력
  - 영상 링크 추가
  - 이미지 업로드 및 관리

### 2. 드래그 앤 드롭 이미지 업로드
```
✨ 사용 방법:
1. 점선 박스에 이미지 파일을 드래그
2. 또는 박스를 클릭하여 파일 선택
3. 여러 이미지를 한 번에 업로드 가능
```

**지원 형식**: PNG, JPG, JPEG, GIF, WEBP

### 3. 이미지 순서 변경 (드래그 정렬)
```
✨ 사용 방법:
1. 업로드된 썸네일의 왼쪽 상단 아이콘을 드래그
2. 원하는 위치로 이동
3. 순서가 자동으로 재정렬됨
```

- **왼쪽 상단**: 드래그 핸들 (⋮⋮)
- **오른쪽 상단**: 삭제 버튼 (X)
- **왼쪽 하단**: 페이지 번호

### 4. E-Book 미리보기
```
✨ 사용 방법:
1. "미리보기" 버튼 클릭
2. 학생 화면처럼 이미지를 넘기며 확인
3. 좌우 화살표로 이동
```

### 5. 자동 제목 생성
- 수업 날짜를 선택하면 자동으로 제목이 생성됩니다
- 예: "2025년 1월 16일 수업"
- 물론 수동으로 수정 가능합니다

### 6. 실시간 피드백
- **이미지 추가**: "N개의 이미지가 추가되었습니다"
- **업로드 중**: 로딩 스피너와 "업로드 중..." 표시
- **업로드 완료**: 성공 메시지와 함께 대시보드로 자동 이동
- **오류 발생**: 명확한 오류 메시지 표시

## 🚀 빠른 워크플로우

### 전체 프로세스 (30초 내 완료!)

1. **학생 선택** (5초)
   - 좌측에서 학생 이름 클릭
   - 또는 검색창에서 검색

2. **날짜 확인** (3초)
   - 기본값이 오늘 날짜
   - 필요시 변경

3. **이미지 드래그** (10초)
   - 바탕화면에서 칠판 이미지들을 선택
   - 드래그하여 점선 박스에 놓기

4. **순서 조정** (5초)
   - 필요시 드래그하여 순서 변경

5. **영상 링크 추가** (5초)
   - ClassIn 링크 붙여넣기
   - 영상 제목 입력 (선택)

6. **발행** (2초)
   - "발행하기" 버튼 클릭
   - 완료!

## 💡 추가 팁

### 효율적인 업로드 방법

1. **여러 학생 연속 업로드**
   ```
   완료 → 좌측에서 다음 학생 선택 → 반복
   ```

2. **이미지 미리 정리**
   ```
   바탕화면에 폴더를 만들고 학생별로 이미지 정리
   → 드래그 앤 드롭으로 빠른 업로드
   ```

3. **키보드 단축키**
   ```
   Tab: 다음 입력 필드로 이동
   Enter: 폼 제출 (이미지 업로드 시)
   ESC: 미리보기 닫기
   ```

### 일반적인 질문

**Q: 이미지를 잘못 올렸어요!**
A: 오른쪽 상단 X 버튼으로 삭제하고 다시 업로드하세요.

**Q: 순서를 바꾸고 싶어요!**
A: 왼쪽 상단의 ⋮⋮ 아이콘을 드래그하세요.

**Q: 미리보기로 어떻게 보일지 확인하고 싶어요!**
A: "미리보기" 버튼을 클릭하면 학생 화면처럼 볼 수 있습니다.

**Q: 영상 링크는 필수인가요?**
A: 아니요, 선택사항입니다. 칠판 이미지만 필수입니다.

**Q: 한 번에 몇 개까지 업로드 가능한가요?**
A: 제한이 없습니다. 필요한 만큼 업로드하세요.

## 🔧 기술 사양

### 사용된 라이브러리
- `react-dropzone`: 드래그 앤 드롭 파일 업로드
- `@dnd-kit/core` & `@dnd-kit/sortable`: 이미지 순서 변경
- `react-hook-form`: 폼 상태 관리
- `@radix-ui/react-toast`: 알림 메시지

### 파일 구조
```
app/admin/classes/new/page.tsx  # 업로드 페이지
components/ui/toast.tsx          # 토스트 알림
components/ui/dialog.tsx         # 미리보기 모달
components/ui/toaster.tsx        # 토스트 컨테이너
```

## 🎨 UI/UX 개선 사항

### Before (기존)
- 단일 컬럼 레이아웃
- 드롭다운으로 학생 선택
- 파일 선택 버튼
- 화살표 버튼으로 순서 변경
- 미리보기 없음

### After (개선)
- ✅ 2단 레이아웃 (학생 선택 + 업로드)
- ✅ 시각적 학생 리스트 + 검색
- ✅ 드래그 앤 드롭 업로드
- ✅ 드래그로 순서 변경
- ✅ E-Book 미리보기
- ✅ 실시간 피드백 (토스트)
- ✅ 자동 제목 생성
- ✅ 로딩 상태 표시
