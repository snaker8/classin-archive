# 멀티 PC 자동 업로드 설정 가이드

다른 센터(지점)의 영상 폴더도 자동으로 모니터링하고 업로드하려면 다음 두 가지 방법 중 하나를 선택할 수 있습니다. 

**💡 추천: 시스템 관리 및 유지보수를 위해 [방법 1: 공유 폴더 기반의 중앙 모니터링]을 강력히 권장합니다.**

---

## 방법 1: 공유 폴더 기반 중앙 모니터링 (추천)
이 방식은 메인 PC 한 대에서 모든 지점의 영상을 관리할 수 있어 유지보수가 압도적으로 편리합니다.

### 1. 클라우드 드라이브 동기화
각 센터(지점)의 영상이 저장되는 폴더를 구글 드라이브, 원드라이브(OneDrive), Dropbox 등의 데스크톱 앱을 이용해 메인 PC로 동기화합니다.
- 예: 센터 A 폴더 -> `D:\OneDrive\센터A_강의실` 
- 예: 센터 B 폴더 -> `D:\OneDrive\센터B_강의실`

### 2. 모니터링 폴더 추가
메인 시스템의 `scripts/monitor-config.json` 파일을 열어 새로 추가된 공유 폴더들의 경로를 `watchDirs` 목록에 추가해 줍니다.

```json
{
  "watchDirs": [
    "D:\\OneDrive\\학원 클래스인 강의실",
    "D:\\OneDrive\\센터A_강의실",
    "D:\\OneDrive\\센터B_강의실"
  ],
  "autoUploadImages": false,
  "autoUploadVideos": true
}
```

### 3. 모니터링 재시작
메인 PC에서 실행 중인 모니터링 창을 닫고 다시 실행하면(`run-monitor.bat` 등) 추가된 폴더들도 함께 감시가 시작됩니다.

---

## 방법 2: 각 PC별 모니터링 개별 설치 (분산 방식)
이 방식은 네트워크 환경이 열악해 메인 PC로의 동기화가 너무 느리거나 불가능할 경우에만 사용하는 것이 좋습니다. 에러 발생 시 해당 PC에 직접 원격 접속하여 확인해야 합니다.

### 필수 준비물
1. **Node.js**: [Node.js 공식 홈페이지](https://nodejs.org/)에서 LTS 버전을 다운로드 및 설치해야 합니다.
2. **.env.local 파일**: 중요 보안 키가 들어있는 파일로, 작동 중인 메인 PC에서 복사해와야 합니다.

## 설정 단계

### 1. 전용 설치 폴더 사용 (보안 강화)
전체 프로젝트 소스 코드를 복사할 필요 없이, 실행에 꼭 필요한 파일만 모아둔 `dist-monitor` 폴더를 사용합니다.

1. **메인 PC**의 `classin-archive` 폴더 안에 새로 생성된 **`dist-monitor`** 폴더를 찾습니다.
(이 폴더에는 핵심 프로그램과 필수 보안 키(`env.local`)만 들어있어 훨씬 가볍고 안전합니다.)

2. 이 `dist-monitor` 폴더를 통째로 복사해서 **새로운 PC**의 원하는 위치(예: `C:\classin-monitor`)에 붙여넣습니다.

### 2. 프로그램 설치 (1회만 수행)
새로운 PC의 `dist-monitor` 폴더 주소창에 `cmd`를 치고 엔터를 눌러 검은색 창을 엽니다.
그 후 아래 명령어를 입력하고 엔터를 누릅니다:

```bash
npm install
```
(필요한 라이브러리가 자동으로 설치됩니다.)

### 3. 자동 실행 등록 (선택)
PC가 켜질 때마다 자동으로 감시가 시작되게 하려면:
**`install-startup.bat`** 파일을 더블 클릭하여 실행합니다.

### 4. 감시 폴더 설정
처음 실행하면 어떤 폴더를 감시할지 물어봅니다. ClassIn이 저장되는 폴더 경로를 입력해주면 됩니다.
(이미 설정되어 있다면 `monitor-config.json` 파일을 수정하거나 삭제하여 재설정할 수 있습니다.)

---

## 작동 확인 방법

1. 새로운 PC에서 **`run-monitor.bat`**를 실행해 봅니다.
2. 검은색 창이 뜨고 파일 감시가 시작되는지 확인합니다.
3. 에러 메시지가 뜬다면 내용을 확인해주세요.

> **주의:** `.env.local` 파일은 업로드 권한을 가진 중요한 키를 포함하므로, 이 폴더가 외부로 유출되지 않도록 주의해 주세요.
