# 학생용 E-Book 뷰어 - 사용 가이드

## 📚 개선된 주요 기능

### 1. **실제 책처럼 넘기는 3D 플립 효과**
```
✨ PC/태블릿:
- 실제 종이책이 넘어가는 듯한 3D 애니메이션
- 두 페이지 동시 보기 (양면 모드)
- 부드러운 페이지 전환 효과

✨ 모바일:
- 한 페이지 보기 (최적화된 뷰)
- 좌우 스와이프로 페이지 넘기기
- 터치 친화적 인터페이스
```

### 2. **고급 확대/축소 기능 (Pan & Zoom)**
```
💡 사용 방법:
1. 더블클릭 → 확대/축소 토글
2. 마우스 휠 → 점진적 확대/축소
3. 모바일: 두 손가락으로 벌리기(Pinch)
4. 확대 후 드래그로 이동

작은 수식도 선명하게! 🔍
```

### 3. **다양한 네비게이션 방법**
```
🖱️ 데스크톱:
- 좌우 화살표 버튼 클릭
- 키보드 화살표 키
- 하단 썸네일 클릭으로 빠른 이동
- F 키로 전체화면

📱 모바일:
- 좌우 스와이프
- 하단 네비게이션 버튼
- 썸네일 스크롤
```

### 4. **수업 영상 연동**
```
🎥 영상 보기 기능:
- 헤더의 "영상 보기" 버튼 클릭
- 새 탭에서 ClassIn 영상 자동 재생
- 원래 탭에서 판서 보며 공부 가능

멀티태스킹 학습! 🎯
```

### 5. **반응형 디자인**
```
💻 PC (1920px+):
- 두 페이지 보기
- 큰 화면으로 편안한 학습
- 키보드 단축키 지원

📱 태블릿 (768-1024px):
- 한 페이지 보기
- 터치 및 마우스 모두 지원

📱 모바일 (<768px):
- 세로 모드 최적화
- 스와이프 네비게이션
- 하단 컨트롤 바
```

## 🎮 조작 방법

### 데스크톱/노트북

#### 페이지 넘기기
- **좌측 화살표 버튼** / **← 키**: 이전 페이지
- **우측 화살표 버튼** / **→ 키**: 다음 페이지
- **썸네일 클릭**: 해당 페이지로 바로 이동

#### 확대/축소
- **더블클릭**: 확대/원래 크기 토글
- **마우스 휠**: 점진적 확대/축소
- **우측 하단 버튼**:
  - 🔍 + : 확대
  - 🔍 - : 축소
  - ↻ : 초기화

#### 기타
- **F 키**: 전체화면 전환
- **Esc 키**: 전체화면 종료

### 모바일/태블릿

#### 페이지 넘기기
- **좌우 스와이프**: 페이지 이동
- **하단 화살표 버튼**: 이전/다음 페이지
- **썸네일 탭**: 페이지 점프

#### 확대/축소
- **더블 탭**: 확대/원래 크기 토글
- **핀치 줌**: 두 손가락으로 벌리기/오므리기
- **확대 후 드래그**: 이미지 이동

## 💡 학습 팁

### 효과적인 복습 방법

1. **첫 번째 읽기: 전체 흐름 파악**
   ```
   - 빠르게 모든 페이지 넘기며 내용 확인
   - 이해 안 되는 부분 체크
   ```

2. **두 번째 읽기: 상세 학습**
   ```
   - 어려운 부분 확대해서 자세히 보기
   - 수식이나 그래프 집중 학습
   ```

3. **영상 연계 학습**
   ```
   - 이해 안 되는 페이지에서 영상 재생
   - 영상 들으며 판서 동시에 보기
   - 반복 학습으로 완전한 이해
   ```

### 학습 시나리오 예시

#### 시나리오 1: 수학 수식 복습
```
1. 페이지를 넘기며 수식 확인
2. 복잡한 수식 부분 더블클릭으로 확대
3. 확대된 상태에서 드래그하며 전체 풀이 확인
4. 이해 안 되면 영상 보기 클릭
5. 영상 들으며 판서 다시 확인
```

#### 시나리오 2: 버스/지하철에서 복습
```
1. 모바일로 접속
2. 세로 모드로 편안하게 보기
3. 스와이프로 빠르게 페이지 넘기기
4. 중요한 부분은 핀치 줌으로 확대
5. 저장하고 싶은 페이지 번호 메모
```

#### 시나리오 3: 시험 전날 총정리
```
1. PC에서 전체화면(F 키) 모드
2. 썸네일 보며 전체 구조 파악
3. 취약한 부분만 집중 학습
4. 영상과 판서를 동시에 보며 최종 정리
```

## 🐛 문제 해결

### Q: 페이지가 넘어가지 않아요
```
A: 
- 화살표 버튼이 비활성화되어 있는지 확인
- 마지막 페이지에서는 다음 버튼이 작동하지 않음
- 새로고침 시도
```

### Q: 확대가 안 돼요
```
A:
- 더블클릭 대신 더블 탭 시도 (모바일)
- 우측 하단 확대 버튼 사용
- 이미지가 완전히 로딩될 때까지 대기
```

### Q: 영상 링크가 작동하지 않아요
```
A:
- 새 탭 팝업이 차단되었는지 확인
- 브라우저 팝업 차단 해제
- 영상 링크가 올바른지 선생님께 문의
```

### Q: 모바일에서 너무 느려요
```
A:
- Wi-Fi 연결 확인
- 다른 앱 종료
- 브라우저 캐시 삭제
```

### Q: 이미지가 흐릿해요
```
A:
- 확대 기능 사용 (더블클릭)
- 전체화면 모드 시도 (F 키)
- 선생님께 고해상도 이미지 요청
```
